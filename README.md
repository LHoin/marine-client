Marine - a Grunt + React + webpack + director webapp framework
================
## 目标
解决SPA（单页面应用）SEO问题

## 关键字
node, react, webpack, director, grunt, ejs, koa, koa-router, JSX, ECMAScript 6

## 初始化项目

1. 安装grunt-cli

		npm install -g grunt-cli

1. 安装项目依赖包

		npm install

	碰到网络不好的时候，依赖包可能安装失败，请删除 `node_modules` 重新安装

1. 运行grunt本地服务器

		grunt server
